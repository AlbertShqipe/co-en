# Concours Entrelacé

**Concours Entrelacé** is a bilingual (French/English) web application built for a dance competition in France. It streamlines candidate registration and admin management for an event that welcomes over 300 participants annually.

## 🩰 Purpose

The goal of this app is to make life easier for the organizers by automating:

- Candidate registrations
- Data management
- Communications
- Payment tracking (if applicable)

Built specifically for the needs of the performing arts world, the app reduces the administrative burden and ensures smooth operation for a high-volume event.

## 🌍 Features

- Bilingual interface (French and English)
- Admin dashboard to manage all candidate data
- Responsive design, accessible on desktop and mobile
- Simple and intuitive registration form for participants
- Email confirmation system for successful registration (optional feature)
- Secure login for admin access

## 🛠 Tech Stack

- **Framework:** Ruby on Rails
- **Frontend:** HTML, SCSS, JavaScript
- **Database:** PostgreSQL
- **Deployment:** Heroku
- **Languages:** French & English
